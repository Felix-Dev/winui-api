# Microsoft.UI.Xaml.Controls.SelectionModel.Deselect

## -description

De-selects the item at the specified index in a 2-level data structure.

## -parameters

### -param groupIndex

The index of the group the item is in.

### -param itemIndex

The index of the item to de-select.
